# AI Voice Assistant

An AI-powered Voice Assistant built using Python, Whisper, Gemini API, Google Text-to-Speech, and Streamlit.

## Features

- Speech Recognition
- Voice Chat
- Text Chat
- AI Responses
- Text-to-Speech
- Modern Streamlit Interface

## Tech Stack

- Python
- Streamlit
- Whisper
- Google Gemini
- gTTS

## Installation

pip install -r requirements.txt

streamlit run app.py
